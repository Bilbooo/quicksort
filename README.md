# Quicksort Visualizer

Visualizzazione interattiva dell'algoritmo **Quicksort** realizzata in HTML, CSS e JavaScript vanilla.

## Demo

Disponibile tramite GitHub Pages: `https://<username>.github.io/quicksort`

## Come funziona

Il progetto anima la scelta del pivot e le partizioni ricorsive dell'array, evidenziando ogni confronto e swap.

## Utilizzo

Apri `index.html` nel browser oppure visita la GitHub Page del progetto.

## Stack

`HTML` · `CSS` · `JavaScript`
